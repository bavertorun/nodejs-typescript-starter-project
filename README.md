# NodeJS & TypeScript Starter Project

## Installation

```
git clone https://github.com/bavertorun/nodejs-typescript-starter-project.git
```

```
cd nodejs-typescript-starter-project
```

```
npm i
```
```
npm run start
```


<br><hr><br>

Start Project Dev:

```
npm run dev
```

Start Project:

```
npm run start
```

Build Project:

```
npm run build
```
